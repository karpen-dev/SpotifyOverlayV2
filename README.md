## SpotifyOverlay v2

Simple Spotify tracks overlay.

**Install Guide:**    
1. Install libs:
    - PkgConfig
    - cpprestsdk
    - nlohmann_json
    - Qt6
2. Register app in [https://developer.spotify.com/](https://developer.spotify.com/)
3. Paste your ``client.id`` and ``client.secret`` into ``config.ini``
4. Run ``./SpotifyOverlay``
5. Allow in browser
6. Play music
